## Run Locally

Clone the project

```bash
  git clone https://github.com/MarceeloDominguez/instagram_clone.git
```

Go to the project directory

```bash
  cd instagram_clone
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
